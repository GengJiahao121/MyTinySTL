# Jiahao Geng's Tiny STL（C++）

## 简介

基于[卡码网KamaCoder](https://kamacoder.com/) **手写简单版本STL** 项目，在其基础上进行容器的迭代器、算法的扩展。并进行对应的**功能测试**和与C++11 STL的**性能测试**。

## 运行

Linux 20.04

编译工具：cmake
